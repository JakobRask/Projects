# Portfolio – Data Science & Machine Learning Projects

Welcome to my GitHub portfolio! This repository showcases a collection of projects covering data analysis, machine learning, deep learning, and business intelligence. Each project demonstrates a complete workflow — from data collection and preprocessing to modeling, evaluation, and insights.


## Projects

### AdventureWorks Business Insights

End-to-end data analytics workflow using Power BI
- Built interactive dashboards using the AdventureWorks dataset
- Performed data cleaning and transformation
- Developed KPIs and business metrics
- Delivered insights for decision-making through visualization

Tools: Power BI, Data Modeling, DAX



### Data Analysis – AdventureWorks 2022

Comprehensive data analysis using SQL and Python
- Queried and analyzed the AdventureWorks2022 database
- Integrated SQL with Python using SQLAlchemy
- Conducted exploratory data analysis (EDA)
- Generated insights through structured queries and visualizations

Tools: SQL, Python, SQLAlchemy, Pandas



### Deep Learning – Facial Expression Recognition

Computer vision project for emotion classification
- Built and compared multiple deep learning models
- Applied convolutional neural networks (CNNs)
- Processed and augmented image datasets
- Evaluated models using accuracy and loss metrics

Tools: Python, TensorFlow / PyTorch, OpenCV



### Machine Learning – Digits Recognition

Handwritten digit classification using the MNIST dataset
- Implemented a complete ML workflow
- Trained and evaluated classification models
- Performed feature scaling and preprocessing
- Compared model performance using standard metrics

Tools: Python, Scikit-learn, NumPy


### Regression Analysis – Car Price Prediction

Predicting car prices using regression models in R
- Combined data from online car listings and SCB API
- Applied regression analysis techniques
- Used best subset selection for feature selection
- Evaluated models using cross-validation and RMSE
- Performed statistical inference on the final model

Key Findings:
- Car price is strongly influenced by age, horsepower, and mileage
- Cars sold by private sellers are generally cheaper than those sold by companies

Tools: R, Regression Analysis, Statistical Modeling
